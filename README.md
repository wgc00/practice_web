###6.Markdown是什么
   Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，
   它可以使普通文本内容具有一定的格式
   Markdown的语法简介明了、学习容易、功能比纯文本更强
   ####Markdown用途
   1.主要用途是编写博客
